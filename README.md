# AI 项目改造器 🤖

> 把热门开源项目改造成CLI工具，打造你自己的AI工具箱！

## 特性

- 🎯 自动获取GitHub热门项目
- 🔧 基于CLI-Anything思路改造项目
- ⚡ 快速生成可执行CLI工具
- 📦 集成热门AI框架

## 已支持

- [ ] browser-use 浏览器自动化
- [ ] mem0 AI记忆层
- [ ] crewAI Agent编排
- [ ] 更多项目待添加...

## 安装

```bash
git clone https://github.com/wallzhou-tech/ai-project-modifier.git
cd ai-project-modifier
pip install -r requirements.txt
```

## 使用

```bash
python main.py --project browser-use
python main.py --list
python main.py --search "AI agent"
```
